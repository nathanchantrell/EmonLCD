LCD Energy Monitor Display using a 4 line LCD display connected with Sparkfun SerLCD Serial Backpack
*****************************************************************************************************

Converted for Arduino 1.0

Builds on emonGLCD by Trystan Lea and Glyn Hudson at OpenEnergyMonitor.org
Adapted for 4 Line Serial LCD by Nathan Chantrell. Hardware details at http://zorg.org/yeg

Uses SoftwareSerial library to allow display to be run one pin other than 2 (now based on NewSoftSerial in Arduino 1.0)

All the required libraries are linked to in code comments.
