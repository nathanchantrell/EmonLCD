LCD Energy Monitor Display using a 4 line LCD display connected with Sparkfun SerLCD Serial Backpack
*****************************************************************************************************

Based on emonGLCD by Trystan Lea and Glyn Hudson at OpenEnergyMonitor.org
Adapted for 4 Line Serial LCD by Nathan Chantrell. Hardware details at http://zorg.org/yeg

Uses NewSoftSerial "soft" serial port library to allow display to be run one pin other than 2.

Builds on OpenEnergyMonitor.org design and code and uses the following libraries (insert into Arduino librarys folder):

JeeLabs Ports: http://jeelabs.net/projects/cafe/wiki/Ports
JeeLabs RFM12B: http://jeelabs.net/projects/cafe/wiki/RF12
OneWire library: http://www.pjrc.com/teensy/arduino_libraries/OneWire.zip
DallasTemperature library: http://download.milesburton.com/Arduino/MaximTemperature/DallasTemperature_371Beta.zip
NewSoftSerial: http://arduiniana.org/libraries/newsoftserial/
